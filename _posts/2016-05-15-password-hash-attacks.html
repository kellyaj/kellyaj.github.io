Password Hashes

A hashing function is a one-way function that implements an algorithm that when given an arbitrary block of data returns a fixed-size bit string called a hash value. Systems that need to store passwords typically store the password hash values, rather than the passwords in clear text. During the authentication process, the password presented by the user is hashed compared with the previously stored message digest.

Password Cracking

Cracking is the process of recovering the clear text passphrase, given the stored hash. Once the hash type is known, a common method of cracking is to simulate the authentication process by repeatedly trying guesses for the password and comparing the newly generated hash with a stolen or dumped hash>

It can be difficult to identify the exact type of hash unless you know the program that was used to create them.

  • The length of the hash (each hash function has a specific output length)
  • The character set used in the hash
  • Any special characters that might be present in the hash
