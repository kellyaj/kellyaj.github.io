A protocol is a set of rules that define how information should be transmitted over a network. The International Organization for Standardization made the Open Systems Interconnections Reference Model -- or OSI Model in hopes of creating a standard for network protocols. The OSI Model divides networking into seven distinct layers:

  1. 7 Application
  2. 6 Presentation
  3. 5 Session
  4. 4 Transport
  5. 3 Network
  6. 2 Data Link
  7. 1 Physical

Each layer communicates with only the layers above and below it. As messages travel from the Application layer to the Physical layer, headers are added at each layer. When a message arrives at a layer, the headers from the previous layer are combined into the data payload. This process is called encapsulation.

At the the Physical layer (1), the message is converted into electrical signals that represent bits and then sent across the wire.
